Get Started

Installation

As this is a purely HTML template, installation couldn't be any easier. Included in the download are two folders (PSD and HTML). To install the template to your website simply upload the contents of the HTML folder to your webserver of choice. This is normally the public_html folder or httpdocs folder. If you aren't sure which folder you need to upload the website to then your webhost should be able to offer you assistance with this.

There should be a number of folders uploaded to your server for the website to display correctly.

Firstly, the 8 HTML pages should be uploaded to the root website folder.

After this there should be the following folders:

  • Images (Contains the websites images)
  • CSS (Contains the Bootstrap framework and website styling)
  • JS (Contains the JQuery files for the slider and accordion)
  • Fonts (Contains the Open Sans font)

Bootstrap Framework

This template relies on the Twitter Bootstrap framework to remain responsive and work correctly on all sized devices. For more information about the framework please visit the Bootstrap Website - Link

The framework is made up of columns equaling to twelve in width. The basic structure should be as follows as you will see throughout the website code:

<div class="container">
    <div class="row">    
        <div class="col-md-9">
            Content
        </div>
        <div class="col-md-3">
            Content
        </div>
    </div>
</div>

The above code will generate a column that stretches 9 and then another column next to it that stretches for 3.

There are a number of different column varities and options such as offsetting that can be viewed in the Bootstrap documentation located here.

Slider Customisation

The website slider allows to you show rotating content including large background images as well as HTML content should you so wish. Included in the index.html file are two sections to control the view of the slider.

The two sections are commented with <!-- #Slider --> and <!-- Mobile Header -->

The reason for the two different versions is to allow the header to display nicely on small mobile phones so make sure you update the content on both to ensure that the content carries across to all device sizes.

Basic Code

<!-- #Slider -->
<div id="sliderWrapper" class="mainSlider">
    <div id="slider" class="nivoSlider">
        <img src="images/header-background.png" data-thumb="images/header-background.png" alt="" title="#htmlcaption" />
        <!-- Place your images here -->
    </div>
    <div class="col-md-12" id="slideControls"></div>
    <div id="htmlcaption" class="nivo-html-caption">
        <div class="container">
            <div class="row">
                <div class="col-md-8 col-md-offset-2">
                    <h1>We provide top end, <strong>high storage virtual servers</strong> at affordable prices.</h1>
                    <p>Lorem ipsum dolor sit amet, consectetur adipiscing elit. Ut vel pharetra diam.
                            Aenean convallis nibh facilisis risus convallis, non ornare tellus vehicula.</p>
                </div>
            </div>
        </div>
    </div>
    <!-- Paste your new slide content code here -->
</div>
<!-- /#Slider -->

The slider is split in to two sections. The area where you place the background images and then if you require the area to add the HTML content.

Adding A New Slide Without HTML Content

Adding a new slide without any HTML content, I.e just an image background is simple. Simply paste the following code in to the section that says <!-- paste your images here -->.

<img src="IMAGESOURCE" data-thumb="IMAGESOURCE" alt="" title="" />

Leave the title value blank as this is what the slider uses to recognise that you want to associate HTML content with the slide.

Adding A New Slider With HTML Content

Adding a new slide with HTML content is just as easy but requires a little bit more work. Firstly, we start out the same as if we were adding a slide without HTML content only this time will fill the title value in like below:

<img src="IMAGESOURCE" data-thumb="IMAGESOURCE" alt="" title="#ContentName" />

Make sure that the title value contains the # tag as this is important and the value also should be case sensitive.

Once you have done that you need to paste some code in after the line that says <!-- Paste your new slide content code here -->

The code to be pasted is below:

<!-- #Caption Content -->
<div id="ContentName" class="nivo-html-caption">
    <div class="container">
        <div class="row">
            <div class="col-md-8 col-md-offset-2">
                <h1>Main Title</h1>
                <p>Text</p>
            </div>
        </div>
    </div>
</div>
<!-- /#Caption Content -->

Make note that we have called the div the exact same name that we placed in the title value previously including the same capital letters. You can then go ahead and edit the test inside the HTML content where the bold values are.

You can add as many slides as you like following the same principle but make sure you choose a different title value for each slide or it will not show the correct HTML content!

Support Accordion

Included on the support page of the template is an accordion to easily hide/show answers to questions. The basic code for the accordion is shown below:

<ul id="faqList">
    <li>Question Title<i class="fa fa-chevron-down fa-lg"></i>
        <p class="faqAnswer closed">Question Answer</p>
    </li>
    <!-- Add a new question below here -->
</ul>

To add a new question simply add a new item to the list, the code which is below.

<li>Question Title<i class="fa fa-chevron-down fa-lg"></i>
    <p class="faqAnswer closed">Question Answer</p>
</li>

Please make sure not to delete the <i> section after the title as this will allow the up and down arrow to show on the right hand side of the question. The same also applied to the class of faqAnswer and closed that are on the paragraph. These are both important to the functionality of the accordion and should be included on all your answers.
